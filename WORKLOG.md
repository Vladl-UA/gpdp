# WORKLOG — рабочий поток между записями журнала STATE.md

Не источник истины и не замена журналу STATE.md (§2 HANDOFF.md). Здесь —
то, что происходит МЕЖДУ его записями: короткие пометки «то сделал, это
понял, то запланировал», без прозы и обоснований, без цели пережить
века. Как только что-то здесь дозревает до настоящего решения — оно
уезжает в STATE.md формальной записью журнала, а не остаётся только
тут. Этот файл может позже подрезаться (старые даты — не архив,
которым дорожат); STATE.md — может.

Пишется в КОНЕЦ (append, не в начало, в отличие от журнала STATE.md).
Читается с конца — `tail`, вверх до даты последней записи в журнале
STATE.md, дальше уже покрыто им напрямую. Формат: `## ГГГГ-ММ-ДД`,
внутри — `ЧЧ:ММ — короткая строка`, без дальнейшей вложенности.

При расхождении с STATE.md — STATE.md главнее (§2 HANDOFF.md): запись
здесь просто не была ещё вовремя туда перенесена, это не альтернативная
версия истины.
